ECL Webcomponents playground

Use the editor to insert the content you want. To include ecl components, click on the ECL button in the toolbar to see all the available components and click on the chosen component's button to get the form where you can edit all the available attributes for that specific component. For simple components you will see in the editor something that will look like the real component, but it's just a way to represent it in a simplified markup, the styles comes from the main ECL website loaded in this page.

ECL components can have a hierarchical structure, a parent and a child, in those case after submitting the parent form the Ui will help you in identifying the component in the editor, you can then start creating children by clicking on the right button and submitting the related button. Children components usually expect content to be inserted, you can start typing it right after submitting the form.

  • If the component supports the content inserted by the user you can type in the editor right after submitting the form
  • To move out of the component press enter
  • If the component has children, once the parent has been created you can start inserting the fist child
  • to move out a children component editing click enter, you can then insert another child
  • components can be edited, to change any of the attributes click on the toolbar button relative to the component you want to edit and submit the form with the updated values.
